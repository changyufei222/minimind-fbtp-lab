# Query Compiler Algorithm Resume Pack

This directory consolidates the most algorithm-facing evidence for the `MiniMind FBBP Query Compiler Lab`.

## Files

- `query_compiler_baseline_matrix.csv`
  - compact machine-readable comparison table
- `query_compiler_baseline_matrix.md`
  - human-readable comparison table plus hardening trajectory
- `query_compiler_error_analysis.md`
  - failure taxonomy, representative examples, and fix path
- `query_compiler_protocol_reliability.md`
  - holdout construction, leakage boundary, and validity note
- `query_compiler_training_card.md`
  - LoRA training details, config summary, and promotion logic

## Suggested use

- Resume writing:
  - use `query_compiler_baseline_matrix.md` and `query_compiler_training_card.md`
- Interview preparation:
  - use `query_compiler_error_analysis.md` and `query_compiler_protocol_reliability.md`
- Quick metric lookup:
  - use `query_compiler_baseline_matrix.csv`
